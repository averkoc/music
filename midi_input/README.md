# MIDI Input Files

Place exported MIDI files from MuseScore Studio here.

These files will be processed by the scripts to add SWAM-compatible CC messages.

## File Naming Convention

Recommended format: `[instrument]_[title]_[version].mid`

Examples:
- `violin_nocturne_v1.mid`
- `saxophone_etude_draft.mid`
- `violin_concerto_movement1.mid`
