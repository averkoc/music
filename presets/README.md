# SWAM Instrument Presets

Save your SWAM Violin and Saxophone presets here for easy recall.

## Preset Types

- `.ssvp` - SWAM Violin presets
- `.sssp` - SWAM Saxophone presets
- `.json` - Custom preset descriptions and CC mappings

## Recommended Presets

Create presets for different styles:
- `violin_classical.ssvp` - Classical vibrato and tone
- `violin_contemporary.ssvp` - Modern, controlled sound
- `saxophone_jazz.sssp` - Jazz articulations
- `saxophone_classical.sssp` - Concert/classical tone
